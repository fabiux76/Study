Configuration

tsconfig.json

File tsconfig.json da creare nella root del progetto (touch tsconfig.json) Tra i parametri principali:

  • include: dir con sorgenty ts
  • lib: es ["es2015"]
  • module: specifica il module system da usare, es commonjs
  • outDir
  • strict
  • target: "es2015"
